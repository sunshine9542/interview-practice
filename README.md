# 인터뷰 연습

말하기·인터뷰 셀프 코칭 연습 웹앱 (PWA). 녹화/녹음은 기기에만 저장됩니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 접속. HTTPS 또는 localhost에서 카메라·마이크 사용 가능.

## 빌드

```bash
npm run build
npm run preview
```

## 주요 기능

- 영상+음성 / 녹음만 연습
- 질문 플래시 + 띵/딸깍 알림음 + 카운트다운
- 답변 시작: 자동 또는 버튼 (설정)
- 리뷰: 재생 + 질문지 + 시점 메모 + 총평
- 다음 연습 전 지난 총평 카드
- 로컬 통계

## 홈 화면에 추가 (PWA)

- **iOS**: Safari → 공유 → 홈 화면에 추가
- **Android**: Chrome → 메뉴 → 앱 설치 / 홈 화면에 추가
